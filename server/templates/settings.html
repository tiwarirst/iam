🤖 Agent Deployment Guide

Deploy agents on target machines to enable remote user management. Each agent communicates with this dashboard via JWT-authenticated REST calls.

Linux Installation
pip install flask PyJWT
sudo python agent/universal_agent.py
Windows Installation (Administrator PowerShell)
pip install flask PyJWT
python agent\universal_agent.py
⚠️ Important: Agents require root/administrator privileges to manage system users. Ensure the JWT secret key matches between server and agent config files.
